# Image Requirements

## Logo
- **File:** `/public/logo.svg` or `/public/logo.png`
- **Format:** SVG (preferred) or PNG with transparent background
- **Dimensions:** Recommended 200x80px or similar aspect ratio
- **Description:** Hexagon with "R" icon, "RESURFACE-IT" text below

## City Landing Page Images
Place the following images in `/public/images/`:

- `city-eugene.jpg` - Aerial view of Eugene, Oregon
- `city-albany.jpg` - Aerial view of Albany, Oregon  
- `city-corvallis.jpg` - Aerial view of Corvallis, Oregon
- `city-springfield.jpg` - Aerial view of Springfield, Oregon

**Recommended specs:**
- Format: JPG or WebP
- Dimensions: 1920x1080px or larger (16:9 aspect ratio)
- File size: Optimized for web (< 500KB each)
- Quality: High resolution for hero backgrounds

## Other Images
- `hero-home.jpg` - Home page hero image (1200x800px recommended)

- `service-{slug}-1.jpg`, `service-{slug}-2.jpg` - Service detail page images
